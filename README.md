# installation

## client
go to the folder of client with 
```
cd client
```
install javascript library with
 ```
 npm install
 ```
install vite in global with
```
npm install -g vite
```

run client side with 
```
vite
```

## api

go to the api folder with

```
cd api
```

install javascript libs with 
```
npm install
```

run api with 
```
npm start
```
